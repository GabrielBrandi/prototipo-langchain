Protótipo de Análise de Documentos com LLM e LangChain
Descrição

Este projeto consiste em um protótipo desenvolvido para permitir que um modelo de linguagem realize a leitura e interpretação de documentos utilizando a biblioteca LangChain. O objetivo é possibilitar a consulta de informações presentes em arquivos de forma natural, por meio de perguntas feitas pelo usuário.

A aplicação processa documentos, extrai seu conteúdo e utiliza um modelo de linguagem para responder perguntas com base nas informações presentes nesses arquivos.

Objetivo

O principal objetivo do projeto é explorar o uso de Modelos de Linguagem (LLMs) integrados ao LangChain para criar uma solução capaz de:

Ler e processar documentos

Indexar o conteúdo dos arquivos

Permitir consultas em linguagem natural

Retornar respostas baseadas no conteúdo dos documentos

Tecnologias Utilizadas

Python

LangChain

Ollama

Modelos de Linguagem (LLM)
